Origin

From Heroku to Cloud Run

Born on 6 October 2015 with a commit message that read "appbot example" — a weekend project to try Node.js, Heroku, and the Telegram Bot API all at once. It shipped with a Bootstrap landing page, an Angular controller, and jquery-2.1.4.min.js.

In 2026 it was resurrected: rewritten in Python, moved into a Bazel monorepo, and deployed on Google Cloud Run. Same jokes API. Different everything else.
